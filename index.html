Common data formats for charities during COVID-19

By adopting common data formats and identifiers across the sector, charities can make their services easier to find, help government make better policy and funders ensure that money is being directed to the right place. This website is a includes practical guides about using different types of data, and lets charities create challenges to agree which standards to use.
