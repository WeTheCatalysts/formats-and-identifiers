Data standards

Common data formats for charities during COVID-19

By adopting common data formats and identifiers across the sector, charities can make their services easier to find, help government make better policy and funders ensure that money is being directed to the right place. This website is a includes practical guides about using different types of data.

Each standard is the result of a challenge set by people working in the sector. If you work with data in the charity sector and want to see a standard way for publishing or exchanging a particular type of data, create a new challenge. You can see the open challenges here.

Adopted standards

Identifiers

Identifiers are standard codes for identifying things like organisations, categories or places.

  • Identify an organisation Proposal
  • Publish or exchange point location data Proposal
  • Identify the nations of the UK that data applies to Proposal

Formats

Formats are standard ways of publishing data about a thing such as a service provided by a charity, or a funding award.

  • Publish or exchange date or time data Proposal
  • Publish information about a service that is provided to the public and the geographical area it covers Proposal
